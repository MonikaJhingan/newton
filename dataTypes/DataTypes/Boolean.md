### Boolean data type

Boolean sbhi programming language ka ek common data type hai.boolean do value ko return krta hai.

*True*



*False*

**Example :-**

```python
a=("")
print(bool(a))
 ```
**Output :-**

`false `

`because empty string is false`

**Example :-**
```python
b=("megha")
print(bool(b))
 ```
**Output :-**
`True `

`because any string is true except empty`

**Example :-**
```python
print(True)
print(type(True))
print(False)
print(type(False))
 ```
**Output :-**

`True`

`<class bool>`

`False`

`<class bool>`
