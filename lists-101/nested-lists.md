## LISTS ki LIST

Yeh list dekhein:

```python
magic_square = [
    [8, 3, 4],
    [1, 5, 9],
    [6, 7, 2]
]

print type(magic_square)
print type(magic_square[0])
print type(magic_square[1])

print sum(magic_square[0])
print sum(magic_square[1])
print sum(magic_square[2])
```
